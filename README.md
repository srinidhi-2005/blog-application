# Blog Management MERN App

This is a simple Blog Management application developed using MERN Stack.

## Technologies Used

- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS

## Features

- Add New Blog
- View All Blogs
- View Single Blog
- Edit Blog
- Delete Blog

## Project Structure

- backend → API, Database, CRUD Operations
- frontend → React User Interface
